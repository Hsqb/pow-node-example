 pow-node-example
====
  

# HOW to Run
## Setup for MAC

````

$ mkdir temp && cd temp

$ git clone https://github.com/Hsqb/pow-node-example.git

$ npm install

````

## Usage
````
$ node main.js [ ASCENT | DESCENT | RANDOM ] [ Difficulty 3 ~ ] [ URL : localhost:4000 ~ 4009 ]

ex) $ node main.js ASCENT 4 localhost:4000
````
